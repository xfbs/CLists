Detailed Description

List node.

Stores some (variable length) data as well as a pointer to the next node in the list.

Invariants

If next is NULL, this is the last node of the list. If next is non-NULL, this is not the last node of the list, and the value of next is a pointer to the next node in the list. All nodes in the list are of the same size (so all elements hold the same amount of data, as measured by the length of data.

Since the list is not circular, next may never point to an element that has already been in the list earlier or to this node.
